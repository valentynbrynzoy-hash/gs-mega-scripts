# 🛠️ GS Mega Scripts — Google Sheets Influencer Data Toolkit

Google Apps Script набор для обработки данных об инфлюенсерах в Google Sheets. Три инструмента в одном файле — дедупликация, сортировка по типу контакта и форматирование ссылок. Имеет простой интерфейс вызывающий по кнопке Мои скрипты. Работает с 2 листами которые структурно по количество, расположение колонок отличается друг от друга. Может быть изменено под другие колонки.

## 📋 Что умеет

### 🧹 Удаление дублей

Ищет одинаковые значения в указанной колонке (по умолчанию — ссылки на профили) и удаляет дубли, оставляя первое вхождение. Работает корректно с ячейками, содержащими формулы `HYPERLINK`.

Упрощает работу при заполнении таблицы когда добавляеш канал который в прошлой уже находил но забыл, таким образом удаляя дубли, так же можно использовать при проверке стороней базы каналов для удаление дублей всей таблицы.

Как описано выше может работать с 2 листами в таблице. Работае отдельно, кнопка удалить дубли связь, удалит дубли в листе связь. 

### 📂 Разделение по типу контакта
Разбирает контактную колонку и раскидывает строки по трём листам:
- **📞 Номера** — есть телефон
- **📧 Почты** — есть email
- **⬜ Без контактов** — пусто

Строки, у которых есть и телефон, и email, попадают в оба листа. Умеет работать с переносами строк, точками с запятой, запятыми в одной ячейке. Полезно когда обрабтал большой обьем данных и их нужно структурировать, функция существенно время сокращает, позволяет сразу скопировать диапазон с почтами и отправить в таблицу для отправки почт ит.д. по списку

Как описано выше может работать с 2 листами в таблице. Работае отдельно, кнопка разделить контакты связь, разделить блок с данными основываясь на диапазоне листа связь. 

### 🔗 Обработка ссылок
Превращает «сырые» URL в кликабельные ячейки с именем профиля в качестве текста. Поддерживает платформы:

| Платформа | Извлекаемое имя |
|-----------|----------------|
| Instagram | `@username` из URL |
| Facebook | `pagename` из URL (кроме `profile.php?id=`) |
| YouTube | `@handle` или `/c/name` |
| TikTok | `@username` |
| Telegram | `username` из `t.me/` |

Ячейки без распознанного имени (например, `/channel/UCxxxxxxx`) оставляются как обычный URL и подсвечиваются в отчёте — доделать вручную. Полезно когда нужно отправить канал агенту для например запроса статистики, удобен тем что позволяет сразу диапазон скопировать ссылок и отпавить в чат месенджера, без необходимости отправлять ссылку на таблицу, обьяснять что где находится или вручную копировать ссылки

## 🚀 Установка

Установить можно двумя способами, либо сделать копию таблицы, как это сделать и ссылку показано в разделе Шаблон таблицы либо вставьте в свою таблицу, как это сделать далее описано. Имейте ввиду что скрипт работает с название стролбцов-название листов как  в  шаблоне, если надо изменить нужно делать правки в скрипт, советую не делать это сам а с помошью claude.ai внести все правки - так меньше риск что то сломать

1. Открой свою Google Таблицу
2. **Расширения → Apps Script**
3. Удали весь дефолтный код
4. Вставь содержимое `Code.gs`
5. Сохрани (`Ctrl+S`)
6. Обнови страницу таблицы — появится меню **🛠️ Мои скрипты**

## 🔐 Авторизация скрипта

При первом запуске любого пункта меню Google попросит дать разрешения.

1. Появится окно **«Требуется авторизация»** → нажми **Продолжить**
2. Выбери Google-аккаунт, к которому привязана таблица
3. Увидишь предупреждение **«Google не проверил это приложение»** — это нормально для личных скриптов
4. Нажми **«Дополнительные настройки»** → **«Перейти на страницу [название проекта] (небезопасно)»**
5. Нажми **Разрешить**

После этого разрешения сохраняются — повторно подтверждать не нужно.

## ⚙️ Настройка под свою таблицу

Все параметры — в объекте `НАСТРОЙКИ` в начале файла. Менять нужно только его, остальной код не трогай.

```javascript
var НАСТРОЙКИ = {

  СВЯЗЬ_ДУБЛИ: {
    НАЗВАНИЕ_ЛИСТА  : "Связь",      // ← имя вкладки в твоей таблице
    КОЛОНКА_ССЫЛОК  : "B",          // ← колонка с URL для проверки
    НАЧАЛЬНАЯ_СТРОКА: 1,            // ← с какой строки начинать (1 = с самого начала)
  },

  // ... остальные блоки по аналогии
};
```

**Структура объекта НАСТРОЙКИ:**

| Блок | Назначение |
|------|-----------|
| `СВЯЗЬ_ДУБЛИ` | Параметры дедупликации для листа "Связь" |
| `ЯНВАРЬ_ДУБЛИ` | Параметры дедупликации для листа "Январь 2026" |
| `СВЯЗЬ_КОНТАКТЫ` | Разделение контактов для листа "Связь" |
| `ЯНВАРЬ_КОНТАКТЫ` | Разделение контактов для листа "Январь 2026" |
| `СВЯЗЬ_ССЫЛКИ` | Обработка ссылок для листа "Связь" |
| `ЯНВАРЬ_ССЫЛКИ` | Обработка ссылок для листа "Январь 2026" |

Если у тебя другие названия листов или колонок — просто поменяй значения в нужных блоках.

## 💡 Нюансы

- **Диапазон строк** задаётся интерактивно в момент запуска (через prompt). Форматы: `5:200`, `50:`, `:100` или пусто = весь лист.
- **Удаление дублей** необратимо — сделай копию листа перед запуском, если не уверен.
- **Разделение контактов** создаёт новые листы (или перезаписывает существующие с теми же именами) — исходный лист не трогается.
- **HYPERLINK-формулы** обрабатываются корректно — извлекается реальный URL, не текст ячейки.

## 🔧 Технические детали

- Runtime: Google Apps Script (V8)
- Зависимости: нет, только встроенный SpreadsheetApp API
- Работает с: Google Sheets (любой тариф, включая бесплатный)
- Язык кода: ES5-совместимый JavaScript (требование Apps Script)
